## Contributing

Every repo has a `README.md` with setup instructions. The general flow:

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Open a PR

Questions? Reach out at [kelson@urantia.dev](mailto:kelson@urantia.dev).
