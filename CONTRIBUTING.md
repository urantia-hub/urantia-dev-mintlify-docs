# Contributing

## Prerequisites

- [Mintlify CLI](https://mintlify.com/docs/development)

```bash
npm i -g mint
```

## Local Development

```bash
mint dev
```

This starts a local preview at `http://localhost:3000`.

## Deployment

Docs auto-deploy on push to `main` via the Mintlify GitHub app. No manual deploy step needed.

## Submitting Changes

1. Fork the repo
2. Create a feature branch (`git checkout -b my-feature`)
3. Run `mint dev` to preview your changes locally
4. Open a pull request against `main`
