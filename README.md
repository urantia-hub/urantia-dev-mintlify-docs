# Urantia.dev Docs

Documentation site for the [Urantia Papers API](https://api.urantia.dev), built with [Mintlify](https://mintlify.com).

## Development

```bash
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.

## Deployment

Changes pushed to the default branch are deployed automatically via the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app).
